# Virtual Clothing Room 👕👔

Um aplicativo web moderno para experimentar roupas virtualmente. Faça upload de uma foto sua, selecione peças de roupa e veja como fica antes de comprar!

## Funcionalidades

- ✅ Upload de foto do usuário
- ✅ Catálogo de roupas (camisetas, calças, jaquetas, vestidos)
- ✅ Sobreposição inteligente de peças
- ✅ Ajuste de tamanho e posição
- ✅ Filtros por categoria e cor
- ✅ Salvamento de looks favoritos
- ✅ Download de imagem final
- ✅ Interface responsiva

## Tecnologias

- **Frontend**: React + TypeScript
- **Estilização**: Tailwind CSS
- **Processamento de Imagem**: Canvas API
- **Estado**: Zustand
- **Build**: Vite

## Instalação

```bash
# Clone o repositório
git clone https://github.com/samsungsturbo900-a11y/virtual-clothing-room.git
cd virtual-clothing-room

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## Estrutura do Projeto

```
src/
├── components/        # Componentes React
├── store/            # Zustand store
├── data/             # Dados do catálogo
├── App.tsx          # Componente principal
└── index.css        # Estilos globais
```

## Como Usar

1. **Upload de Foto**: Clique em "Upload" e selecione uma foto sua
2. **Escolher Roupa**: Navegue pelo catálogo de roupas
3. **Ajustar**: Use os controles para redimensionar e reposicionar
4. **Salvar**: Guarde seus looks favoritos
5. **Baixar**: Exporte a imagem final
